# Images Directory

This directory contains all images for the Brassband Eensgezindheid website.

## Directory Structure

- `gallery/` - Photos from performances and events
- `events/` - Event-specific images
- `brassband-hero.jpeg` - Main hero image for the homepage
- `gertjan-loot.jpeg` - Photo of conductor Gertjan Loot
- `favicon.ico` - Website favicon

## Image Guidelines

- All images should be optimized for web use
- Use descriptive filenames
- Include alt text for accessibility
- Maintain consistent aspect ratios where possible
